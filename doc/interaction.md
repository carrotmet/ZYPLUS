# 职业规划网站交互设计

## 核心交互功能

### 1. 专业选择器（主页核心功能）
- **树形结构导航**：左侧显示本科专业目录的层级结构
  - 第一层：学科门类（工学、理学、文学等）
  - 第二层：专业类（计算机类、电子信息类等）
  - 第三层：具体专业（计算机科学与技术、软件工程等）
- **交互流程**：
  - 用户点击学科门类，展开专业类
  - 点击专业类，展开具体专业列表
  - 点击专业名称，右侧显示该专业的详细信息卡片
- **搜索功能**：顶部搜索框支持专业名称关键词搜索，实时筛选结果

### 2. 职业路径可视化（专业详情页）
- **职业卡片展示**：以卡片形式展示该专业对应的职业方向
- **晋升路线图表**：使用流程图展示从初级到高级的职业发展路径
- **薪资趋势线图**：使用ECharts展示不同经验年限的平均薪资变化
- **交互功能**：
  - 点击职业卡片查看详细要求和发展前景
  - 悬停显示该职业的详细信息弹窗
  - 切换不同职业路径进行对比

### 3. 信息添加系统（信息管理页）
- **添加专业信息**：
  - 表单包含：专业名称、学科门类、专业简介、主要课程、学制年限
  - 实时预览添加的专业卡片效果
- **添加职业信息**：
  - 表单包含：职业名称、所属行业、学历要求、技能要求、薪资范围
  - 关联专业选择器，可关联多个相关专业
- **数据验证**：实时验证表单输入，确保数据完整性

### 4. 智能推荐系统
- **基于专业推荐职业**：根据选择的专业自动推荐相关职业
- **职业匹配度显示**：用百分比和星级显示专业与职业的匹配程度
- **相似专业推荐**：当用户选择某个专业时，推荐相关的其他专业

### 5. 个人经历信息模块（新增）
- **个人信息表单**：
  - 昵称：用户自定义显示名称
  - 专业：选择所属专业
  - 学历：学士/硕士/博士
  - 学校名称：毕业或就读学校
  - 学位：具体学位名称
  - 个人经历：详细的工作经历和职业发展现状
- **经历展示**：以时间线形式展示个人职业发展轨迹
- **经验分享**：用户可分享职业经验和建议
- **匿名显示**：保护用户隐私，可选择匿名展示

### 6. 新增信息显示功能（本次开发重点）
- **专业信息实时显示**：用户添加专业信息后，立即显示在主页专业树形结构中
- **职业信息实时显示**：用户添加职业信息后，立即显示在职业详情页的选择器中
- **个人经历信息实时显示**：用户添加个人经历后，立即显示在新增的"经历分享"界面中
- **数据同步机制**：前端与后端数据实时同步，确保显示内容最新
- **动态UI更新**：无需刷新页面，动态更新相关界面内容

## 用户交互流程（更新版）

1. **首次访问**：用户进入主页，看到专业选择的树形导航界面
2. **专业探索**：通过点击或搜索找到感兴趣的专业
3. **职业了解**：查看该专业的职业发展路径和薪资信息
4. **信息贡献**：用户可以添加新的专业或职业信息，完善数据库
5. **个人经历分享**：用户可添加个人职业经历，帮助他人了解真实职业情况
6. **实时反馈**：新增信息立即在相应界面显示，提供即时反馈
7. **深度对比**：可以收藏多个专业/职业进行对比分析

## 响应式设计

- **桌面端**：左侧树形导航 + 右侧内容区域的经典布局
- **移动端**：折叠式导航菜单 + 全屏内容展示的移动优化布局
- **平板端**：侧边栏可收缩的混合布局

## 动画效果

- **页面切换**：使用Anime.js实现平滑的页面过渡动画
- **数据加载**：骨架屏和渐进式内容加载动画
- **交互反馈**：按钮点击、卡片悬停的微交互动画
- **图表动画**：ECharts图表的数据加载和更新动画
